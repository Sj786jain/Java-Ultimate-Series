## 📅 Daily Projects  

### ✅ Completed Projects  

- **Day 1: Hello REST API**  
  A simple Spring Boot REST API that returns a greeting message.  
  Practiced `@RestController`, `@RequestMapping`, `@GetMapping`, and basic response handling.

- **Day 2: To-Do List API**  
  CRUD To-Do API using in-memory ArrayList.  
  Practiced `@PostMapping`, `@PutMapping`, `@DeleteMapping`, query params, and response entity handling.

- **Day 3: Student Management API**  
  Spring Boot CRUD API for managing students (create, update, delete, list).  
  Practiced Spring Data JPA basics with H2 database.

- **Day 4: Course Management API**  
  CRUD API for managing courses with student-course relationships.  
  Practiced entity relationships & `@OneToMany` / `@ManyToOne`.

- **Day 5: Book Store API**  
  CRUD API for managing books, authors, and inventory.  
  Practiced DTOs, service layer, and validation.

- **Day 6: Combined API (Students + Courses + Books + Users)**  
  Integrated multiple APIs into one application.  
  Practiced modular package structure and combined endpoints.
